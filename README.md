# CORRE GOODS — 001 · A Rua é o Ponto

Apresentação estratégica (landing page em HTML) do projeto **CORRE GOODS**: social run + DJ set em movimento + after run em São Luís — MA.

> **running / music / street**

A primeira edição é um **piloto de ~100 pagantes em meados de outubro**: 6km leve de manhã na orla, DJ acompanhando o percurso e after run monetizado (bebidas, comida leve, produtos e ativação 18+). Estética off-white / streetwear, território sonoro **100% eletrônico**.

## Arquivos

- **`corre-goods-001-apresentacao.html`** — a apresentação. Abra no navegador (`F11` = tela cheia). Funciona offline.
- **`corre-goods-001-apresentacao.pdf`** — versão PDF para enviar.
- **`GUIA-APRESENTACAO.md`** — guia de uso, edição de textos/imagens e checklist.
- **`assets/`** — identidade visual e as 9 fotos da marca (otimizadas para web).
- **`imagens/`** — fotos originais em alta resolução.
- **`docs/pesquisa-estrategica-corre-goods.md`** — relatório estratégico de base.

## Como apresentar

Abra o HTML, role de cima para baixo seguindo a narrativa (13 seções: tese → oportunidade → como funciona → edição 001 → marca → público → modelo de negócio → after run → ativações → São Luís → plano → proposta → encerramento). Os botões do topo saltam direto para cada bloco.
